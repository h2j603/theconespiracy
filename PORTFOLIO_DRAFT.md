# The Conespiracy

전 세계에서 수집한 트래픽콘(라바콘) 사진을 국가별로 모아 보여주는 정적 아카이브 웹사이트.

## 서술

세계 각지의 트래픽콘 사진을 한 곳에 모아 기록하는 단일 페이지 웹사이트다. 프레임워크나 빌드 도구 없이 HTML·CSS·바닐라 JavaScript만으로 구성했고, 사진은 국가 섹션(한국·일본·태국·중국·독일)으로 나뉘어 각 섹션을 `<details>`/`<summary>`로 접고 펼치게 했다. 안내 문구와 제목은 한국어·영어·일본어·독일어·중국어·태국어로 병기되며, 상단에는 CSS 애니메이션으로 흐르는 마키 문구를 두었다. 사진을 클릭하면 캡션과 함께 확대 프리뷰가 페이지에 표시되도록 `main.js`에서 클릭 핸들러를 붙였고, 제보는 인스타그램을 통해 받는다.

## 스펙

- **역할**: 기획·디자인·프론트엔드 개발 (1인)
- **스택**: HTML, CSS, JavaScript (바닐라, 빌드 도구 없음)
- **주요 의존성**: 외부 라이브러리·프레임워크 없음. 이미지는 Amazon CloudFront CDN URL로 직접 참조
- **구조**: `index.html`, `style.css`, `main.js` 3개 파일로 구성된 정적 단일 페이지
- **주요 기능**: 국가별 `<details>` 접이식 갤러리 / 이미지 클릭 확대 프리뷰(`main.js`) / 다국어(6개 언어) 텍스트 병기 / CSS 마키·페이드 애니메이션
- **링크**: 리포 — `github.com/h2j603/theconespiracy` · 라이브 — (미확인)
